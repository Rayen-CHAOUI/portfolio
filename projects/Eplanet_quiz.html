Eplanet Quiz

Project Overview

Eplanet Quiz is an interactive English quiz platform inspired by the TCF exam. It is designed to help users practice English comprehension through audio, image, and text-based questions while tracking performance.

Key Features

  • User registration and login system (with ID and password)
  • Multiple difficulty levels: Beginner, Intermediate, Advanced
  • Interactive quiz interface with audio and image support
  • Ability to change answers before clicking “Next”
  • Result page with detailed table of correct/wrong answers

Admin Panel

  • View registered users and their performance
  • Add new quiz questions manually
  • Import quiz data from JSON or Excel files
  • Upload related images and audio files

Technologies Used

  • Frontend & UI: Flet (Flutter-like UI in Python)
  • Backend: Python, SQLite3
  • Assets: Local image/audio files for questions
  • SQLite for user data and quiz questions

Project Structure

  • backend/: Authentication and database logic
  • frontend/: All UI views (login, signup, dashboard, quiz, results, admin)
  • assets/: Image and audio folders for quiz media
