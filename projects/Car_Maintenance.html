Car Maintenance

Car Maintenance est une application mobile développée avec Flutter (Dart) pour aider les utilisateurs à suivre l'entretien de leurs véhicules. Elle utilise SQLite comme base de données locale pour stocker les informations des voitures et de leurs entretiens.

Fonctionnalités

  • Ajouter un ou plusieurs véhicules avec les informations suivantes :
    • Marque
    • Modèle
    • Année
  • Visualiser la liste de tous les véhicules enregistrés
  • Sélectionner un véhicule pour afficher la liste de ses entretiens
  • Ajouter un nouvel entretien avec :
    • Type d'entretien (vidange, révision, etc.)
    • Description
    • Date de début
    • Date d'expiration (facultative)
    • Prix
    • Kilométrage actuel
    • Prochain kilométrage prévu pour vidange

Expérience utilisateur

  • Au premier lancement, l’utilisateur peut :
    • Choisir un véhicule existant
    • Ou en créer un nouveau
  • Une fois un véhicule sélectionné, les entretiens s'affichent dans une page dédiée
  • Un bouton flottant en bas à droite permet d’ajouter facilement un nouvel entretien

Technologies utilisées

  • Flutter / Dart : développement mobile
  • SQLite : gestion locale des données
