Finger Counter

Une application Python interactive qui détecte les doigts levés en temps réel avec webcam et interface graphique

Fonctionnalités

Détection & Comptage

  • Détection en temps réel jusqu’à deux mains simultanément
  • Comptage précis du nombre de doigts levés

Interface Graphique Moderne

  • Interface avec CustomTkinter
  • Lancement en un clic
  • Mode clair et sombre
  • Affichage de l’heure en direct
  • Flux webcam intégré dans la fenêtre

Technologies Utilisées

Python 3
OpenCV
MediaPipe
CustomTkinter
