Finger Counter

Finger Counter est une application Python interactive qui utilise OpenCV et MediaPipe pour détecter les mains et compter le nombre de doigts levés en temps réel. Elle est dotée d'une interface graphique élégante et moderne conçue avec CustomTkinter.

Fonctionnalités

  • Détection en temps réel jusqu’à deux mains simultanément.
  • Comptage précis du nombre de doigts levés pour chaque main.
  • Interface graphique interactive avec :
    • Lancement en un seul clic.
    • Mode clair et sombre.
    • Affichage de l’heure en temps réel.
    • Flux vidéo via la webcam intégré à l’interface.

Technologies utilisées

  • Langage : Python 3
  • Vision par ordinateur : OpenCV
  • Détection des mains : MediaPipe
  • Interface graphique : CustomTkinter
