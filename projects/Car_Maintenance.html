Car Maintenance

Application mobile Flutter pour la gestion intelligente de l’entretien des véhicules

Fonctionnalités

  • Ajouter un ou plusieurs véhicules :
    • Marque
    • Modèle
    • Année
  • Visualiser tous les véhicules enregistrés
  • Afficher les entretiens d’un véhicule
  • Ajouter un entretien avec :
    • Type (vidange, révision...)
    • Description
    • Date de début & expiration
    • Prix
    • Kilométrage actuel & prévu

Expérience Utilisateur

  • Au premier lancement :
    • Sélection d’un véhicule existant
    • Ou création d’un nouveau
  • Affichage des entretiens du véhicule
  • Ajout rapide via bouton flottant

Technologies Utilisées

  • Flutter / Dart pour le développement mobile
  • SQLite pour la gestion locale des données
