Battleship Arcade Game (Bataille de Navale)

Un jeu web interactif de Bataille Navale développé avec HTML, CSS, JavaScript pour le frontend, et Node.js, Express.js, MongoDB pour le backend. Cette application propose une expérience riche pour les joueurs ainsi qu’un tableau de bord administrateur complet pour la gestion et l’analyse.

Fonctionnalités

Tableau de Bord Administrateur

  • Gestion des Joueurs :
    • Affichage de tous les joueurs enregistrés
    • Ajout manuel de nouveaux joueurs
    • Suppression de joueurs de la base de données
  • Journaux d'Activité : Suivi et visualisation des actions réalisées par les utilisateurs
  • Rapports Statistiques : Génération de rapports visuels et graphiques sur les activités et performances des joueurs
  • Barre de Navigation : Liens vers Joueurs, Logs, Ajouter un joueur, Supprimer un joueur, Rapports

Tableau de Bord Joueur

  • Création de compte et connexion :
    • Inscription avec nom d’utilisateur et adresse email uniques
    • Connexion sécurisée
    • Réinitialisation de mot de passe via email
  • Modes de Jeu : Jouer contre la machine, Multijoueur local, Multijoueur en ligne avec Socket.IO
  • Classement des joueurs : Affichage en direct des scores

Technologies utilisées

  • Frontend : HTML, CSS, JavaScript
  • Backend : Node.js, Express.js
  • Base de Données : MongoDB
  • Multijoueur temps réel : Socket.IO
